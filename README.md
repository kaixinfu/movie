## Install

> `git clone git@github.com:liukaixin01/movie.git`

> `npm install`

## mongodb

> `下载连接mongodb`

> `连接mongodb：$ mongod --config /usr/local/etc/mongod.conf`


> `连接不上请查看：/usr/local/var/log/mongodb/mongo.log 日志`

> `$ mongo`
